- ! Exclamation Mark

- \# Hashtag/Pound symbol
- $ Money sign
- % Percent 
- ^ Caret
- & Ampersand/And symbol
- \* Asterisk or Multiplication symbol
- ( Open bracket
- ) Close bracket
- _ Underscore
- \+ Plus or addition symbol

Shortcuts I would like to start using:

- Ctrl-Enter (adds www. and .com to an URL)
